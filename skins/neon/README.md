# Neon Cat Skin

A vibrant cyberpunk neon-themed Bongo Cat skin.

## Customization

To customize this skin:

1. Replace the PNG files with your own neon-styled cat images:
   - `cat-rest.png` - Idle/resting pose
   - `cat-left.png` - Left paw slapping
   - `cat-right.png` - Right paw slapping

2. Update `skin.json` with your information:
   - Change the `author` field to your name
   - Update the `description` if you modify the theme
   - Adjust `rotation_degrees` if needed (-13 is default)

3. Select this skin in Bongo Cat:
   - Right-click cat → Settings → Skin → Neon Cat

## Image Requirements

- **Format**: PNG (with transparency recommended)
- **Size**: Any size (will be scaled automatically)
- **Style**: Match the neon aesthetic (glowing edges, vibrant colors, cyberpunk theme)

## Style Tips

For authentic neon aesthetics:
- Use bright, saturated colors (cyan, magenta, yellow)
- Add glowing outlines or edges
- Consider dark backgrounds with bright accents
- Use gradient effects for a neon sign look

## Sharing Your Custom Skin

If you create a custom neon skin you'd like to share:
1. Create a new folder in `skins/` with your theme name
2. Add your images and a `skin.json` file
3. Consider sharing it in the GitHub discussions!
